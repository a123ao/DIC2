# 強化學習 (Reinforcement Learning) HW1: 網格地圖與價值迭代

這是一個使用 Python Flask 與 HTML/CSS/JS 開發的互動式網頁應用程式，用於展示強化學習中的網格世界 (Grid World) 環境、策略評估 (Policy Evaluation) 以及價值迭代 (Value Iteration) 演算法。

## 作業要求與功能實作

### HW1-1: 網格地圖開發 (Grid Map Development)
- **動態網格生成**：支援產生 $n \times n$ 的網格地圖，允許用戶自訂維度 $n$ (範圍為 5 到 9)。
- **互動式環境設定**：
  - **起始點**：用戶可透過滑鼠點選指定起始單元格 (標示為綠色)。
  - **終點**：用戶可透過滑鼠點選指定結束單元格 (標示為紅色)。
  - **障礙物**：設定好起終點後，用戶可依規定點選設定 $n-2$ 個障礙物單元格 (標示為灰色)。

### HW1-2: 策略顯示與價值評估 (Policy Display & Value Evaluation)
- **隨機行動策略**：地圖上會顯示每個單元格的隨機生成行動 (以上、下、左、右箭頭圖示表示)。
- **策略評估**：針對當前的隨機策略，推導並在方格上顯示每個狀態的價值函數 $V(s)$。

### HW1-3: 使用價值迭代演算法推導最佳策略 (Value Iteration for Optimal Policy)
- **價值迭代實作**：在後端實作價值迭代 (Value Iteration) 演算法來計算環境的最佳政策。
- **最佳政策顯示**：計算完成後，前端會以箭頭顯示出每個狀態對應的最佳行動，取代先前的隨機行動。
- **價值函數更新**：同步更新並顯示收斂後的價值函數 $V(s)$，代表在最佳政策下各狀態的期望回報。

## 系統架構
- **後端 (Backend)**：使用 `Flask` 框架負責處理路由、初始化網格狀態，以及執行強化學習演算法 (Policy Evaluation & Value Iteration)。
- **前端 (Frontend)**：使用 `HTML`/`CSS`/`JavaScript` 刻劃使用者介面，透過網頁互動 (如滑鼠點擊) 來建構地圖並以視覺化的方式觀察各種策略與價值變化。

## 執行與使用方式

1. **進入專案目錄與啟動虛擬環境**
   專案已內建 `.venv` 虛擬環境，請在終端機執行以下指令啟動：
   ```bash
   # Windows 系統
   .\.venv\Scripts\activate
   ```

2. **安裝所需套件** (如未安裝)
   ```bash
   pip install Flask
   ```

3. **啟動 Flask 伺服器**
   ```bash
   python app.py
   ```

4. **開啟網頁進行操作**
   - 伺服器啟動後，請開啟瀏覽器並前往 `http://127.0.0.1:5000` (或終端機提示的本機網址)。
   - 在網頁上輸入指定的網格大小 $n$ (5~9)，點擊生成地圖。
   - 按照畫面說明，依序用游標點擊設定：**起點 (綠) $\rightarrow$ 終點 (紅) $\rightarrow$ 障礙物 (灰)**。
   - 觀察隨機策略的狀態價值，並透過介面按鈕執行價值迭代，查看各狀態更新後的最佳行動箭頭與期望回報 $V(s)$。
